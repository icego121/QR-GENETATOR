# QR-GENETATOR

สร้าง QR Code จากลิงก์ที่คุณต้องการ ใส่โลโก้ตรงกลางได้ และเลือกระดับ Error Correction เพื่อให้สแกนได้แน่นอนแม้มีโลโก้บัง — ทำงานแบบ **client-side ล้วน ๆ** ไม่มีการส่งข้อมูลใด ๆ ออกจากเครื่อง และใช้งานได้แม้ไม่มีอินเทอร์เน็ต (ไฟล์ไลบรารีและฟอนต์ถูกดาวน์โหลดมาไว้ในโปรเจกต์แล้ว)

พัฒนาขึ้นเพื่อใช้งานเอง และเปิดให้ทุกคนใช้งานฟรี รวมถึงนำไปแก้ไข/พัฒนาต่อได้ตามสัญญาอนุญาต MIT

## ฟีเจอร์

- กรอกลิงก์แล้วสร้าง QR Code ขนาด 500×500px
- แนบรูปโลโก้ตรงกลาง QR ได้ (ขนาด 30% ของ QR พร้อมมุมโค้ง)
- เลือกระดับ Error Correction ได้ 4 ระดับ (L / M / Q / H) — ล็อกที่ H อัตโนมัติเมื่อมีโลโก้ เพื่อให้สแกนติดแน่นอน
- ดาวน์โหลดผลลัพธ์เป็น PNG หรือ SVG
- ทำงาน 100% ในเบราว์เซอร์ (client-side only) ไม่ต้องมีเซิร์ฟเวอร์ ไม่ต้องเชื่อมต่ออินเทอร์เน็ต

## วิธีใช้งาน

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไรเพิ่ม

```bash
git clone https://github.com/<your-username>/qr-genetator.git
cd qr-genetator
# เปิด index.html ด้วยเบราว์เซอร์
```

## โครงสร้างโปรเจกต์

```
QR-GENETATOR/
├── index.html
└── assets/
    ├── css/style.css       # สไตล์หน้าเว็บ
    ├── js/script.js        # ลอจิกสร้าง/ดาวน์โหลด QR
    ├── vendor/             # ไลบรารี qrcode.js (vendored ไว้ใช้งาน offline)
    └── fonts/              # ฟอนต์ที่ใช้ (vendored ไว้ใช้งาน offline)
```

## เทคโนโลยีที่ใช้

- HTML / CSS / JavaScript (vanilla)
- [qrcode](https://github.com/soldair/node-qrcode) สำหรับสร้าง QR Code
